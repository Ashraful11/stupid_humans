Overview

Every Monday morning, marketing teams waste hours manually pulling LinkedIn analytics into reports. Page views, engagement rates, follower growth - all copied one by one from LinkedIn's analytics dashboard into spreadsheets.

This tutorial shows you how to automate the entire process using Google Apps Script. Once set up, you'll wake up every Monday to a fresh analytics report in your inbox.

Step 2: Connect to LinkedIn API

Now we'll set up the LinkedIn API credentials to pull your analytics data.

1

Create LinkedIn Developer App

  1. Go to LinkedIn Developers
  2. Click Create App
  3. Fill in your app details (use your company page)
  4. Select "Sign In with LinkedIn" and "Share on LinkedIn" permissions

Client Secret: your_client_secret_here

Step 3: Write the Apps Script

This is where the magic happens. We'll write a simple script to fetch LinkedIn data.

1

Open Apps Script editor

  1. In your Google Sheet, click Extensions ‚Üí Apps Script
  2. Delete any existing code in the editor
  3. Paste the code below
Code.gs 
function fetchLinkedInAnalytics() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();

  // Your LinkedIn API credentials
  const CLIENT_ID = 'YOUR_CLIENT_ID';
  const CLIENT_SECRET = 'YOUR_CLIENT_SECRET';
  const ACCESS_TOKEN = 'YOUR_ACCESS_TOKEN';

  // LinkedIn API endpoint
  const url = 'https://api.linkedin.com/v2/organizationalEntityShareStatistics';

  // Set up API request
  const options = {
    'method': 'get',
    'headers': {
      'Authorization': 'Bearer ' + ACCESS_TOKEN,
      'Content-Type': 'application/json'
    }
  };

  try {
    // Make API call
    const response = UrlFetchApp.fetch(url, options);
    const data = JSON.parse(response.getContentText());

    // Extract metrics
    const pageViews = data.totalShareStatistics.shareCount || 0;
    const engagement = data.totalShareStatistics.engagementCount || 0;
    const impressions = data.totalShareStatistics.impressionCount || 0;

    // Calculate engagement rate
    const engagementRate = impressions > 0 ?
      ((engagement / impressions) * 100).toFixed(2) : 0;

    // Get current date
    const today = new Date();
    const formattedDate = Utilities.formatDate(
      today,
      Session.getScriptTimeZone(),
      'yyyy-MM-dd'
    );

    // Append data to sheet
    sheet.appendRow([
      formattedDate,
      pageViews,
      engagementRate + '%',
      data.followerGain || 0,
      impressions,
      data.clickThroughRate || 0
    ]);

    Logger.log('Analytics updated successfully!');

  } catch (error) {
    Logger.log('Error: ' + error.toString());
  }
}

What this code does

  1. Connects to LinkedIn API using your credentials
  2. Fetches analytics data for your company page
  3. Calculates engagement rate automatically
  4. Adds new row to your Google Sheet with today's date

Step 4: Schedule the automation

Set up a trigger to run this script automatically every Monday at 8 AM.

1

Create a time-based trigger

  1. In the Apps Script editor, click the clock icon (Triggers) in the left sidebar
  2. Click Add Trigger (bottom right)
  3. Configure the trigger:
    • Function: fetchLinkedInAnalytics
    • Event source: Time-driven
    • Type: Week timer
    • Day: Monday
    • Time: 8am to 9am
  4. Click Save

Troubleshooting

Error: "Authorization required"

Solution: Your access token may have expired. LinkedIn access tokens last 60 days. Generate a new one from the LinkedIn Developer portal and update your script.

No data appearing in the sheet

Solution: Check these common issues:

  • Verify your CLIENT_ID and CLIENT_SECRET are correct
  • Make sure you have admin access to the LinkedIn page
  • Check the Apps Script logs: View ‚Üí Logs
  • Test the function manually first (click the Run button)

Script runs but shows zeros

Solution: LinkedIn's API requires specific permissions. Go back to your LinkedIn app settings and ensure "r_organization_social" permission is enabled.
